## Description

Api rest for aws-s3 files management.

## Installation

```bash
# If not installed:
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Database

If not installed:
https://docs.docker.com/compose/install/

```bash
# running mongo db from docker:
$ docker-compose up
```
